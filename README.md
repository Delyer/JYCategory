# **扩展工具：JYExtension**


**工具简介：** 整理项目中常用的category和方法. 提升开发效率. 持续更新.........

**主要功能：**

- 数据的加密以及数据转换
- 数组安全操作
- 处理打印时unicode编码问题，打印出正常显示的中文字符串
- Base64编码
- 二维码生成（支持渐变颜色和logo）和条形码生成
- 线程方法
- URL编码
- 沙盒文件管理
- 本地化存储
- UIView的事件处理以及坐标转换
- 十六进制生成颜色类以及随机颜色
- 图片处理以及生成
- 设备信息的获取
- 等等。。。。。。。
- 持续更新。。。。。。
